# Advent of Code

Solutions for advent of code in multiple languages. Each year is solved in a different language.

- 2015 - [Python](https://www.python.org/)
- 2016 - [Typescript](https://www.typescriptlang.org/) with [Bun](https://bun.sh)
- 2017 - [Go](https://go.dev/)
- 2018 - [Rust](https://www.rust-lang.org/)
- 2025 - [Gleam](https://gleam.run/)
