# AoC 2025 - Python

Advent of code 2025 problems solved using [Gleam](https://gleam.run/).

## Installation and setup

- [Install Gleam](https://gleam.run/getting-started/installing/)

## Running

Each script in `src/` folder corresponds to a single AoC problem. Part 1 and part 2 of each problem
are implemented in separate files. All scripts take input from stdin and print the answer to stdout.

```sh
gleam run -m day_01_p1 < input.txt
```

You can get inputs from [AoC 2025](https://adventofcode.com/2025).

> NOTE: This year only has 12 problems
